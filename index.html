JOSHUA HULL
Twitter

Untangling the resourceful web with Renee

During this talk, Joshua will discuss how the request model differs between Sinatra and Rails, and how we can use this to create captivating Web Applications without repeating ourselves. He will also delve int internals, and take a look at real-world applications written in Renee. Lastly, we'll understand how to write extensions to Renee.

Joshua Hull has been working as a programmer for the past 12 years. He's deeply familiar with the world of startups, most recently through his involvement with companies such as Bibliocommons, Postrank and Poll Everywhere. Joshua is currently employed by Twitter; when he isn't working, he stays busy with open source projects such as Padrino, Bundler, and whatever other crazy ideas occur to him.

Twitter page

DAN CROAK
Thoughtbot

Volunteers, not employees

Good programmers can work from anywhere, pick their projects, and deploy their work globally in an instant. Demand outstrips supply.

What if employers considered their employees highly paid volunteers? This talk will examine the questions programmers and programmer-employers ask ourselves when choosing projects and teammates.

Dan has been making things on the web for 15 years. He's "a pretty good programmer with great habits."

Personal page

ALEX MACCAW
Twitter

JavaScript Web Applications and Asynchronous Interfaces

Alex MacCaw is mainly a Ruby and JavaScript developer, specializing in creating JavaScript RIAs. He is competent in both backend, frontend and design.

Alex has spoken internationally at various conferences (RailsConf, GoRuCo) as well as closer to his home, in London. Alex just wrote a book on building

JavaScript Web Applications for O'Reilly. Last year he set up a startup in San Francisco called Taskforce.

Personal page

MARC-ANDRÉ COURNOYER
Macournoyer.com

The 4 Hour Work Week for Ruby Hackers

Marc-André Cournoyer (coor · noah · ye), a very friendly code/life hacker from beautiful Montréal, QC.

He created Thin, tinyrb, wrote the book Create Your Own Programming Language, created the group chat application Talker. And he's now making a living online teaching Rails & programming languages.

Personal page

COREY HAINES
Code Retreat

Keynote

Corey Haines started programming at the age of 10, on a computer his father (also a programmer) bought for him. He played lots of computer games, and when he couldn't win the games, he learned to read the source code and edit the outcome. These "winning games" were his first software programs. He continued to hack, grew up, and began working as a professional software developer.

PostgreSQL

Harold Giménez is a maker of clean, well tested, maintainable code that is delivered fast. He specializes in Machine Learning, Ruby/Ruby on Rails, and PostgreSQL. Harold is currently a Software Engineer at Heroku.
